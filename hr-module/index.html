Scope
What’s Covered
All features included in this prototype with current status.
👤
Employee Directory Prototype Full list with EMP ID, name, branch, department, designation. Add / Edit / View profile. Filters by region, branch, department, designation, status.
✅
Daily Attendance Prototype Mark Present / Absent / Half-Day / On Duty per employee per day. Branch-wise view. Bulk mark. Auto-carry for holidays and leaves.
🌎
Leave Management Prototype Apply and approve CL, SL, EL, LOP, Maternity, Comp Off, On Duty. Year-to-date balances. Leave calendar per employee.
🕐
Shift Management Prototype 5 shift templates covering all branch clusters. In/Out times, break duration, working hours calculation. Branch-to-shift mapping.
📅
Holiday Calendar Prototype 2026 holidays with type (National / State / Optional) and location scope. Tamil Nadu state holidays included. Election-day overrides.
📊
Attendance Reports Prototype Monthly attendance summary by employee or branch. Present / Absent / LOP / Leave day counts. Exportable to Excel.
📋
Leave Reports Prototype Leave utilisation by type, department and branch. Remaining balance per employee for current financial year.
📆
Pay Period Config Prototype 26th–25th pay cycle definition. Period lock/unlock, cutoff date management, payroll-period list view.
🛡
Role-based Access Planned HR Admin, Branch Manager, Employee self-service roles. Manager approval workflow for leaves and attendance corrections.
📱
Mobile / PWA Planned Employee self-service: apply leave, view payslip, check attendance from mobile browser or app shell.
💰
Payroll Integration Planned Export attendance + LOP data per pay period to payroll. Integration hooks with existing Rosca finance tables.
🔔
Notifications Planned Leave approval alerts via WhatsApp / email. Birthday and anniversary reminders for HR. Regularisation reminders.
